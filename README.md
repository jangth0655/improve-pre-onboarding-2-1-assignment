# 과제 개선해보기

- 프리온보딩 교육과정에서 진행 하였던 기업과제를 학습한 내용을 바탕으로 개선해보았습니다.

## 🏁 프로젝트 실행 방법

1. 의존성 패키지를 설치합니다.

```zsh
$ npm install
```

2. 프로젝트를 실행합니다.

```zsh
$ npm start
```
